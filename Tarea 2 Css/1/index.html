Ejercicio de selectores y propiedades de texto

Objetivo:

El objetivo de este ejercicio es que aprendas a utilizar los selectores CSS para seleccionar elementos HTML.

Más información sobre selectores en CSS AQUÍ

Lo primero que debes hacer es FORK de este documento. 💾 Recuerda guardar tus cambios para no perderlos.

Instrucciones:

Entra en "styles.css" y añade lo necesario para poder hacer lo siguiente: (recuerda que solo podrás tocar CSS no puedes añadir nada en el HTML)

  • Utiliza el selector "h1" para cambiar el color del texto del título a rojo y tamaño 20px.
  • Utiliza el selector "p" para cambiar el tamaño de la fuente del párrafo a 16px.
  • Utiliza el selector "a" para cambiar el color de fondo del enlace a azul.
  • El "h2" debe ser de color verde y tamaño 18px.
  • El "h2" del "section " debe ser de color azul.
  • El "a" del "article " debe ser de color negro y fondo rojo.
  • El "p" del "article" con la clase "párrafo" debe ser de color amarillo
  • Los "a" del "nav" deben ser de diferentes colores
  • El "footer" tiene que tener el fondo morado
  • En el "footer" El primer párrafo debe ser blanco y el segundo amarillo
